# AppAnvil
A GUI for the Linux Security Module AppArmor

## Dependencies
### Compile Time
* cmake
* g++ (or another equivalent c++ compiler)
* libgtkmm-3.0-dev
### Optional
* glade
* xmllint

## Compilation Instructions
To generate a makefile, and build the project using that makefile:
```
$ cmake .
$ make
```
## Run
To run the resulting binary:
```
$ ./dist/appanvil
```
